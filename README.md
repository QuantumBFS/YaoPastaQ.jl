To install, in the Julia REPL do

```julia
julia> add YaoPastaQ
```

For usage:

```julia
julia> using Yao, YaoPastaQ
julia> YaoPastaQ.genlist(put(3, 3=>Y))
1-element Vector{Any}:
 ("Y", 3)
 
julia> x = PastaQReg(bit"00110")
PastaQReg{ITensors.MPS}
    active qubits: 5/5

julia> nqubits(x)
5
```
